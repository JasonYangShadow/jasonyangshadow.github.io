Sort Colors

Given an array with n objects colored red, white or blue, sort them in-place so that objects of the same color are adjacent, with the colors in the order red, white and blue.

Here, we will use the integers 0, 1, and 2 to represent the color red, white, and blue respectively.

Note: You are not suppose to use the library’s sort function for this problem.

Example:

Input: [2,0,2,1,1,0] Output: [0,0,1,1,2,2] Follow up:

A rather straight forward solution is a two-pass algorithm using counting sort. First, iterate the array counting number of 0’s, 1’s, and 2’s, then overwrite array with total number of 0’s, then 1’s and followed by 2’s. Could you come up with a one-pass algorithm using only constant space?

The first solution is scanning all data firstly then set the value in position. but it also requires o(m+n) space, as the generated list requires space

class Solution:
    def sortColors(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        num0 = 0
        num1 = 0
        num2 = 0
        #constant space, scan the count of target number
        for num in nums:
            if num == 0:
                num0 += 1
            if num == 1:
                num1 += 1
            if num == 2:
                num2 += 1
        
        nums[0:num0-1] = [0]*num0
        nums[num0:(num0 + num1 -1)] = [1]*num1
        nums[num0 + num1:] = [2]*num2

The second solution is moving item one by one in place without swap.

class Solution:
    def sortColors(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        n0 = -1
        n1 = -1
        n2 = -1
        
        for num in nums:
            #if num = 0, from the left side, we need to move every item by 1
            if num == 0:
                n2 += 1
                n1 += 1
                n0 += 1
                nums[n2] = 2
                nums[n1] = 1
                nums[n0] = 0
            if num == 1:
                n2 += 1
                n1 += 1
                nums[n2] = 2
                nums[n1] = 1
            if num == 2:
                n2 += 1
                nums[n2] = 2

class Solution:
    def sortColors(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        red, white, blue = 0, 0, len(nums) -1
        while white <= blue:
            #if nums[cur] == 0, we have to move both cur and start
            if nums[white] == 0:
                nums[white], nums[red] = nums[red], nums[white]
                white += 1
                red += 1
            elif nums[white] == 1:
                white += 1
            else:
                #if nums[cur] == 2, we have to swap current item with last unclassified one
                nums[white], nums[blue] = nums[blue], nums[white]
                blue -= 1

        #make sure using if ... elif.. else rather than 3 if, it will cause unexpected behavior
