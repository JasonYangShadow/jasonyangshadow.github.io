migration to hexo

At last, my self-host website migrates from wordpress to hexo, which cost me several hours to make everything work as expected. Wordpress is good, easy to build but too heavy for me, so many plugins and have to expose wp-login.php to public with security risks. Hexo is much lighter, using markdown as editor so I could use my vim. Though, it is a bit of difficult to deploy static files to self-host server, self-host git and git hooks help me so much so that I could edit, generate and deploy my website anywhere. Also hexo makes the structure of my wewbsite clearer, nginx is used as web server and ufw is enabled. Through port mapping feature of my wireless router, everything goes well and the migration is easy to complete.

Love Hexo!!
